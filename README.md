# WhatsApp Web Clone

Projeto desenvolvido com o objetivo de recriar a interface do WhatsApp Web utilizando apenas HTML e CSS.

##  Sobre o projeto

Este projeto consiste na construção de uma cópia visual da interface do WhatsApp Web, com foco em:

- Estruturação semântica com HTML5
- Posicionamento com CSS (fixed, absolute, relative)
- Organização de layout
- Estilização moderna
- Prática de UI Clone

O objetivo principal é praticar fundamentos de front-end e aprimorar habilidades de desenvolvimento de interfaces.

##  Tecnologias utilizadas

- HTML5
- CSS3

##  Objetivos de aprendizado

- Trabalhar com layout fixo
- Criar sidebar fixa
- Utilizar bordas e arredondamentos
- Melhorar organização de código
